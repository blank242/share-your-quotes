# Share Your Quotes

SillyTavern third-party extension that turns copied text or chat messages into a polished quote image you can save as PNG.

## Features

- Adds a `Save quote image` action to each message
- Opens a quote composer from the Extensions settings panel
- Can read directly from the clipboard
- Saves selected quote text, title, author, and save time in the extension settings JSON, with a synced `share-your-quotes.json` backup in SillyTavern files
- Lets you review saved quotes, delete them, or reopen them in the image composer
- Renders quote cards in multiple built-in styles inspired by reading apps
- Saves the result locally as a PNG file
